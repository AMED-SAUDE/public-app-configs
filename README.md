# public-app-configs

Configuracoes publicas consumidas por aplicativos cliente.

## Infecto Assist Ionic

Arquivo de configuracao:

- `configs/infecto-assist-ionic.json`

URL publica (raw):

- `https://raw.githubusercontent.com/AMED-SAUDE/public-app-configs/main/configs/infecto-assist-ionic.json`

Campos aceitos no JSON:

- `subtitle` (string)
- `heading` (string)
- `selectPlaceholder` (string)
- `openButtonLabel` (string)
- `hint` (string)
- `privacyPolicyUrl` (string, http/https)
- `environments` (array de objetos com `id`, `name` e `url`)
